

# 图的存储

设图的节点数为 n, 边数为 m

## 邻接矩阵(稠密图, 节点少, m 多 n 少)

```C++
g[N][N] 	// N 为节点个数
g[a][b] 表示 a 到 b 的有向边的长度
    
无向图中 g[a][b] = g[b][a], g[N][N] 是对称矩阵
```

## 邻接表(稀疏图, 节点多, m 少 n 多)

### 数组模拟链表

```C++
int h[N], e[N], ne[N], w[N], idx;
// h[N] 记录节点 1~n 节点的邻接链的头节点的 idx

memset(h, -1, sizeof h);
// 相当于 Node* h[N], 然后整个 h[N] 初始化为 nullptr

// e[N], ne[N], w[N] 对应一般的节点定义中的结构体 
struct Node
{
    int num;	// 结点编号 b
    int val;	// a->b 边的权重
    Node* next;	// 下一结点的地址 a 的下一邻接点的地址, 头插法的话就是 head[a]
}
// p 对应 idx
// p->num 对应 e[idx]
// p->next 对应 ne[idx]
// p->val 对应 w[idx]

void add(int a, int b, int v)
{
    e[idx] = b, ne[idx] = h[a], w[idx] = v, h[a] = idx++;
}
// 等价于
auto p = new Node()	// idx
p->num = b			// e[idx] = b
p->next = head[a]	// 头插法 ne[idx] = h[a]
p->val = v			// w[idx] = v
head[a] = p			// h[a] = idx++
```

**纯净版**

```C++
int h[N], e[N], w[N], ne[N], idx;

memset(h , -1, sizeof h);

void add(int a, int b, int v)
{

```

### 链表

```C++
struct Node
{
    int num;	// 结点编号
    int val		// 结点的值, 含义是边的权重
    Node* next;	// 下一结点的地址, 含义是下一邻接点的地址
}

Node* h[N]; // 初始化为 nullptr

void add(int a, int b, int v)
{
    auto p = new Node()
    p->num = b;
    p->val = v;
    p->next = h[a];	// 头插法
    h[a] = p;
}
```
