# 双指针算法

双指针算法思想更重要, 没有固定的模板形式

需要满足的性质: 

j 具有单调性:

​	i 往后走的时候, j 不会往前走, 也是往后走

```C++
// i, j 同向

// j 在 i 之前
for(int i = 0, j = 0; i < n; i++)
{
    while(j < i && check(j)) j++;
}

// j 在 i 之后
for(int i = 0; i < n; i++)
{
    int j = i + 1;
    while(j < n && check(j)) j++;
    
    i = j - 1;
}

// i, j 相向
for(int i = 0, j = n - 1; i < j; i++)
{
    while(i < j && check(j)) j--;
}
```

```C++
遍历元素
    插入尾
```
