# 第一讲 基础算法总结

## 二分算法

```C++
bool check(int x)
    
// 右分界点
// 二分性质: 右半边为 true, 左半边为 false
int l, r;
while(l < r)
{
    int mid = l + r >> 1;
    if(check(mid)) r = mid;
    else l = mid + 1;
}
return l;

// 左分界点
// 二分性质: 左半边为 true, 右半边为 false
int l, r;
while(l < r)
{
    int mid = l + r + 1 >> 1;
    if(check(mid)) l = mid;
    else r = mid - 1;
}
return l;

// check(l) 为 true : l 是结果
// check(l) 为 false: l 不是结果
```
