## 数论

### 质数

**质数判定**

```C++
bool is_prime(int x)
{
    if (x < 2) return false;
    for (int i = 2; i <= x / i; i ++ )
        if (x % i == 0) return false;
    return true;
}
```

**质因数分解**

```C++
void divide(int x)
{
    for(int i = 2; i <= x / i; i++)
    {
        if(x % i == 0)
        {
            int cnt = 0;
            while(x % i == 0) x /= i, cnt++;
            cout << i << " " << cnt << endl;
        }
    }
    if(x > 1) cout << x << " " << 1 << endl;
}
```

**筛质数**

朴素筛法

```C++
for(int i = 2; i <= n; i++)
{
    if(!st[i])
        for(int j = i + i; j <= n; j += i) st[j] = true;
}
```

**线性筛法**

```C++
int primes[N], cnt;
bool st[N];

void get_primes(int n)
{
    for(int i = 2; i <= n; i++)
    {
        if(!st[i]) primes[cnt++] = i;
        for(int j = 0; primes[j] <= n / i; j++)
        {
            st[primes[j] * i] = true;
            if(i % primes[j] == 0) break;
        }
    }
}
```
