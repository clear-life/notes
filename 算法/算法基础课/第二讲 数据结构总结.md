# 第二讲 数据结构总结

## STL

**大小比较**

```C++
less<type>()	// 前是否小于后
greater<type>()	// 前是否大于后
    
vector<type> v	// 数组
sort(v.begin(), v.end(), less<type>());		// 数组从小到大排序
// less<type>() less 类重载 () 运算符, less<type>() 不再是创建临时对象, 而是当作一个比较函数
sort(v.begin(), v.end(), greater<type>());	// 数组从大到小排序

priority_queue<type> q;		// 优先队列, 堆, 默认大顶堆
priority_queue<type, vector<type>, less<type>> q;	// 显式构建大顶堆
priority_queue<type, vector<type>, greater<type>> q;// 显式构建小顶堆

// 大顶堆父节点大于子节点, 要用 less, 因为 less(新插入节点, 父节点)
// 小顶堆父节点小于子节点, 要用 greater, 因为 greater(新插入节点, 父节点)
```
