# 位运算

## 概念

### 异或

$\oplus$

相同为 0, 不同为 1

```C++
0 ^ 0 = 0
0 ^ 1 = 1
1 ^ 1 = 1
```

任意一个数 a

1. **自身与自身异或为 0**

   $a \oplus a = 0$

2. **任意数与 0 异或不变**

   $a \oplus 0 = a$

3. 交换律

   $a \oplus b = b \oplus a$

4. 结合律

   $a \oplus b \oplus c = a \oplus (b \oplus c) = (a \oplus b) \oplus c$
