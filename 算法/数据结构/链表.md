# 链表总结

## 基础

```C++
// 结点定义
struct Node
{
	int val;
    Node *next;
    Node(int x): val(x), next(nullptr) {}
};

// 插入
// 将结点 b 插入到 a c 之间
Node a, b, c;
a->next = c;

b->next = c;
a->next = b;

// 头插法
Node head, new_head;
new_head->next = head;
head = new_head;
```

**遍历到第 k 个结点**

```C++
// 假定下标从 0 开始
// 则长度为 len 的链表最后一个结点的下标为 len - 1

// 遍历到下标为 k 的结点
auto p = head;
for(int i = 0; i < k; i++) p = p->next;
```

> 循环不变式:  p 指向下标为 i 的结点
>
> 证:
>
> 初始化: i = 0  p = head 成立
>
> 保持: 
>
> 假定第 i 轮(循环变量为 i)循环开始前, 循环不变式成立, 即 p 指向下标为 i 的结点
>
> 第 i 论循环中: 
>
> p = p->next	则 p 指向下标为 i + 1 的结点
>
> i 自增后 p 还是指向下标为 i 的结点
>
> 则第 i 轮循环结束后, 循环不变式成立
>
> 终止: 循环结束时, i = k, 则 p 指向下标为 k 的结点

## 模板

### 通用

**结点个数**

```C++
int n = 0;
for(auto p = head; p; p = p->next) n++;		// head 到 尾部空结点的跳数
```

**跳转指定次数到指定结点**

```C++
auto a = start;
for(int i = 0; i < k; i++) a = a->next;		// start 到 目标节点 相差 k 步
```

**反转指定个数的结点**

```C++
auto a = start, b = start->next;
for(int i = 0; i < k; i++)		
{
    auto c = b->next;
    b->next = a;
    a = b, b = c;
}
start->next = nullptr;

// 从 start 结点开始反转链表, 直到目标节点(次数为 start 到目标节点的距离, 也就是 k)
// 最终 a 走到目标节点, start 到目标节点中间的链表都反转了, 如果有需求, 还得手动把 start->next 设为 nullptr
```
