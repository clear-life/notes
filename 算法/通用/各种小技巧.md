### 多维数据一维化展开

**三维数据**

(i, j, k) 展开为 (i * j 的维度 + j) * k 的维度 + k

```C++
m * n * l 三维数组
(i, j, k) 展开为 (i * n + j) * l + k
```

### 离散化

哈希函数可以利用 STL 中的 `unordered_map` 帮我们实现

但具体的转换需要我们来做

为什么呢?

因为哈希函数只认一维数据, 如果我们想把一个多维数据离散化到 0~n 的话, 就需要将其一维化

```C++
unordered_map<type, int> hash;
type get(data)	// 将多维数据转化为可以哈希的一维数据
{
    ...
}

for(int i = 0; i < n; i++)
{
    hash[get(data[i])] = i;
}
```
