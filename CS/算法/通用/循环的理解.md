# 循环的理解

## 基础理解

### for 循环

**1. 遍历 $[st, ed)$ 元素**

```C++
for(int i = st; i < ed; i++)
    ...
// 遍历 [st, ed) 的所有元素
```

**2. 重复 k 次**

```C++
for(int i = 0; i < k; i++)
    ...
// 相当于遍历 [0, k) 的所有元素
```

### while 循环

**1. 遍历 $[st, ed)$ 元素**

```C++
int i = st;
while(i != ed)
{
    ...
    i++;
}

```

**2. 重复 k 次**

```C++
while(k--)
{
    ...
}
```
