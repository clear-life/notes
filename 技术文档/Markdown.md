# Markdown



* 标题前后空一行
* 标题结尾不要有标点符号



### 推荐的结构

前后都有空行



| # 标题      |
| :---------- |
| 作者        |
| 摘要        |
| 目录        |
| ## 标题1    |
| ### 标题1.1 |
| ## 标题2    |
| ### 标题2.1 |
| ### 标题2.2 |



## 段落

* 空白行是分隔段落的标志
* 段内换行:结尾插入两个及以上的空格后回车
* 每行不可太长



## 列表

* 嵌套列表:table 键
* 列表项之间没有换行:1个空格
* 列表项之间有换行:无序列表3个空格,有序列表2个空格
* 列表项没有换行,则列表项之间不换行
* 列表项有换行,则列表项之间换1行
* 列表项前后分别空1行
